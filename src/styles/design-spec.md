# Claudio 音乐播放器设计规格

## 1. Concept & Vision

**Claudio** 是一个个人 AI 电台 — 像一位老朋友式的 DJ，在深夜爵士酒吧的氛围中为你播报音乐。界面传达温暖、私密、有深度的音乐体验，而非冰冷的流媒体服务。视觉语言借鉴黑胶唱片、复古收音机和现代 hi-fi 设备。

**关键词**: 深夜电台、黑胶质感、温暖琥珀、DJ 个性

---

## 2. Design Language

### 色彩系统
```css
--bg-deep: #0d0d0f;           /* 深黑背景 */
--bg-surface: #151518;         /* 卡片/表面 */
--bg-elevated: #1c1c21;        /* 悬浮元素 */
--amber-glow: #e8a54b;         /* 主管 - 琥珀暖光 */
--amber-soft: #c4863a;        /* 次要琥珀 */
--amber-dim: #8b5a20;          /* 暗部琥珀 */
--text-primary: #f0ede6;       /* 主文字 - 米白 */
--text-secondary: #8a8680;      /* 次要文字 */
--text-muted: #4a4844;         /* 弱化文字 */
--accent-red: #d64545;         /* 强调红 - 播放状态 */
--vinyl-black: #1a1a1d;        /* 黑胶唱片色 */
```

### 字体
- **Display/标题**: "Cormorant Garamond" (衬线，优雅、编辑感)
- **Body/UI**: "DM Sans" (现代，清晰)
- **数字/时间**: "JetBrains Mono" (等宽，技术感)

### 动效
- **黑胶旋转**: 当前播放歌曲有缓慢旋转的唱片视觉
- **脉冲光晕**: 播放按钮有琥珀色呼吸光晕
- **渐入**: 列表项 staggered fade-in (80ms 间隔)
- **悬浮反馈**: 按钮/卡片 hover 时微微上浮 + 光晕增强

## 4. Components

### PlayerCard (核心播放卡片)
- 状态: 播放中 / 暂停 / 无歌曲
- 唱片封面带旋转动画 (playing 时旋转，pause 时停止)
- 播放/暂停按钮带脉冲光晕

### SearchBar
- 圆角输入框，暗色背景
- 搜索图标，实时搜索建议
- 按 Enter 搜索，回车响应

### PlaylistItem
- 序号 / 封面缩略图 / 歌曲信息 / 时长 / 喜爱按钮
- Hover 显示操作按钮 (播放/移除)
- 当前播放歌曲高亮

### PlayControls
- 上一首 / 播放暂停 / 下一首 / 随机 / 循环
- 进度条可拖拽

## 5. Technical Approach

- **状态管理**: Zustand store
- **音乐 API**: NeteaseCloudMusicApi (公开接口)
- **音频**: HTML5 Audio API
- **样式**: CSS Modules + CSS Variables
- **字体**: Google Fonts (Cormorant Garamond, DM Sans, JetBrains Mono)
