Features

核心功能

不只是播放器，而是一个能理解你口味、记住你偏好的音乐伙伴。

🎙

AI DJ 播报

结合时段和你的口味，AI 像电台 DJ 一样自然地介绍下一首歌。

🧠

自主记忆

自动学习你的音乐偏好、习惯和不喜欢的类型，跨会话持久化，无需手动设置。

💬

自然语言控制

"放点轻松的歌"、"音量调到30"、"下一首放周杰伦" — 说人话就行。

🎵

歌词同步

LRC 歌词逐行滚动，支持翻译歌词对照显示，全屏沉浸模式。

🎨

动态主题

从专辑封面提取主色调，进度条、按钮、背景渐变随音乐实时变化。

📋

歌单管理

创建本地歌单，一键导入网易云歌单，支持 800+ 首大歌单优化加载。

AI System

AI 系统架构

两层意图路由 — 简单指令直接执行，语义请求走 AI 理解。

1

意图分类

正则匹配简单指令（播放/暂停/下一首）→ 直接执行，无需 AI

↓ 语义请求
2

上下文构建

注入记忆、播放状态、时段、用户品味、对话历史

↓
3

LLM 调用

发送到 Claude / MiniMax API，返回结构化 JSON（say + actions）

↓
4

Action 执行

解析 JSON 中的 play / queue / player / playlist 指令，执行对应操作

↓ 静默
5

记忆提取

自动分析对话，提取用户偏好/习惯/不喜欢，写入持久化记忆

Getting Started

快速开始

克隆项目，配置 API 密钥，一键启动。

# 克隆仓库 git clone https://github.com/yourname/claudio.git cd claudio # 安装依赖 npm install # 配置环境变量（编辑 .env 填入 API key 和网易云 cookie） cp .env.example .env # Tauri 桌面端开发 npm run tauri dev # 构建 Windows 安装包 npm run tauri build
